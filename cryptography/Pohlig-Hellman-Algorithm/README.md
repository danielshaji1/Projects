# Pohlig-Hellman Algorithm

This folder contains a notebook-based cryptography lab focused on solving discrete logarithm problems in groups whose order factors into small primes.

## Topics Covered

- Exponentiation by squaring
- Baby-step giant-step
- The Pohlig-Hellman algorithm in a simple case
- The Pohlig-Hellman algorithm in the general case

## Files

- `Pohlig_Hellman_algorithm.ipynb`: the main notebook with explanations, exercises, and implementations

This project highlights why Diffie-Hellman parameters need carefully chosen group orders with large prime factors.
