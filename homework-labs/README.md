# Homework And Labs

This folder contains homework, lab, and assignment projects grouped from multiple college courses.

## Included Projects

- `CS-HW-6`: Python homework working with CSV-based car data
- `CSC-HW-4`: mixed Python coursework files, attendance exercises, and lab practice
- `HW5_src_DanielShaji_732`: restaurant food ordering simulation in Python
- `Lab13`: C networking lab with build and run instructions
- `Wk05a-HW02-Lists`: custom list data-structure assignment in Java
- `Wk05e-FlowerQuiz`: Java flower quiz application
- `Wk07a_Sorting`: Java sorting assignment and driver
- `Wk08a-HW03`: stack and queue project with report materials
- `Wk15b-HW05-Tree-Builder`: tree-builder project with AVL, BST, and red-black tree code
- `csc-lab-exam`: Python lab exam submission

These folders were kept close to their original course submission names so the work is easier to trace back to each class assignment.
